About Me

UI/UX Designer

Halo, nama saya Muhammad Nouval. Saya adalah siswa dari SMKN 4 Bandung, jurusan Rekayasa Perangkat Lunak. Mengapa saya memilih jurusan tersebut? Karena saya sangat menyukai teknologi, terutama yang berkaitan dengan game. Saya sangat senang bermain game, sampai-sampai saya pernah membuat game sederhana meskipun hasilnya belum terlalu bagus. Selain itu, saya juga pernah membuat sebuah website e-commerce dari awal, mulai dari katalog, navbar, hingga keranjang belanja. Saya membuat kode tersebut sendiri menggunakan Vue.js dan juga mengimplementasikan sistem pembayaran dari keranjang belanja tersebut.
Pengalaman membuat website e-commerce memberikan saya wawasan yang lebih luas mengenai pengembangan web. Dari membuat katalog produk, merancang navigasi yang intuitif, hingga mengembangkan sistem keranjang belanja, semuanya saya lakukan sendiri menggunakan Vue.js. Implementasi sistem pembayaran dari keranjang belanja tersebut juga merupakan tantangan yang menambah pemahaman saya tentang integrasi dan keamanan dalam aplikasi web. Semua pengalaman ini memperkaya pengetahuan saya dan mengukuhkan pilihan saya untuk mendalami Rekayasa Perangkat Lunak.
